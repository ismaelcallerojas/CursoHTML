Listas

Las listas nos sirven para enlistar elementos de un página web

Tenemos 2 tipos de listas: ordenadas y no ordenadas: Por ejemplo:

Las listas ordenadas (ol) tienen una secuencia de números:

  1. Elemento
  2. Elemento
  3. Elemento
  4. Elemento

Las listas no ordenadas (ul) no tienen una secuencia de números:

  • Elemento
  • Elemento
  • Elemento
  • Elemento

Las listas tambien pueden ser combiandas o anidadas:

  • Elemento
    1. Elemento Interno
    2. Elemento Interno
    3. Elemento Interno
  • Elemento
  • Elemento
    1. Elemento Interno
    2. Elemento Interno
    3. Elemento Interno
  • Elemento
